Informacion sobre el navegador

Este navegador fue creado con la intencion de mejorar la experiencia a la hora de ver anime. Ya que en algunos navegadores los videos van demaciado lentos, se paralizan constantemente, y esto se debe a que el navegador utiliza demasiados recursos de tu computadora para poder ejecutarlo. NPVA es el navegador ideal para que puedas tener una mejor experiencia a la hora de ver Anime, esta hecho a base de Chormium y tiene solo lo necesario para reproducir un video dentro de una de las paginas para ver anime.
